# dotfiles

## Installation

### symbolic link
```bash
git clone git@github.com:st-8/dotfiles.git
chmod +x ~/dotfiles/link.sh
source ~/dotfiles/link.sh
```

### Homebrew
```bash
cd ~/dotfiles/homebrew
brew bundle
```

### Python
```bash
pyenv install 3.10.5
pyenv global 3.10.5
pip install -r ~/dotfiles/python/requirements.txt
```
